How it works

From solar input
to operator output

01 · Ingest
Live feeds
Real-time NOAA SWPC and NASA OMNI: 1-min Kp, X-ray, IMF Bz, solar wind, proton flux, GloTEC TEC grid, plus 3-day forecast.
Refresh every 5 min · Archive 30 days
02 · Model
Physics + ML
324-cell global grid runs Klobuchar/Mannucci, CCIR-888, Bailey PCA, Nakagami SATCOM. A trained ridge model adds 24-hour Kp forecasts.
324 cells · 5 horizons · auditable JSON weights
03 · Translate
Operator answers
Outputs you can act on: GPS error in meters, HF outage in minutes, route segments to avoid, recommended fallback comms.
<200 ms · SHA-256 audit trail · Bearer-token API
